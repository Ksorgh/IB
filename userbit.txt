Специальные биты управления доступом
1. userbit - устанавливается только для файлов
Позволяет запустить исполняемый файл от имени владельца 
Необходимо для программ, выполняющихся от имени обычных пользователей, которые изменяют обьекты файловой системы, защищённые от записи для всех категорий пользователей.

2. Установка userbit (в режиме администратора):

sudo chmod u+s <имя исп. программы>
sudo chmod u+s passwd

3. Выводы команды stat для программы passwd:
Устройство: 8/4 Инода: 3150126 Ссылки: 1
Доступ: (4755/-rwsr-xr-x) Uid: (0/ root) Gid: (0/rc
Доступ: 2025-09-16 20:00:00.040668071 +0700
Модифицирован: 2024-05-30 00:00:00.000000000 +0700
Изменён: 2024-10-14 11:50:00.422894761 +0700
Создан: 2024-10-14 11:55:00.219894749 +0700 (время примерное т.к работаю на своей машине без ssh сервера)

4. В случае установки s-бита владельца для файла, у которого не было прав доступа на выполение, s-бит выводится в верхнем регистре
stat /home/student/1.txt
Access: (4664/-rwSrw-r--)

5. Сброс user-bit (в режиме адм):
sudo chmod u-s <имя исп. программы>
