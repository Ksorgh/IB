Установка группового бита (на файлы и на папки)
sudo chmod g+s <имя обьекта> 
sudo chmod g+s script.bash
Установка g-бита на файл позволяет запускать файл от имени группы
sudo chmod g+s Fold1
Установка g-бита на папку позволяет наследовать группу владельца на все подпапки и файлы, распаложенные внутри
Смена группы владельцев у папки Fold1:
sudo chown :group1 Fold1
mkdir Fold1/Fold_test создает папку с группой владельцев group1 и наследует g-бит от родительской папки
touch Fold1/test.txt создает файл с группой владельцев group1. g-бит не наследуется файлом

Пример прав доступа к файлу с g-битом:
2664/rw-rwSr--

Пример прав доступа к папке c g-битом:
2775/rwxrws-x

Установка sticky-бита (на папки)
sudo chmod o+t <имя папки>
По умолчанию файлы созданые владельцем внутри папки могут удалять либо владельцы либо члены группы владельцев
Папки с установленным t-битом запрещают удаление файлов всем кроме владельцев
при этом при разрешении на запись можно удалить любую информацию, находящуюся внутри такого файла.
sudo chmod o+t Fold3

Пример доступа к папке с t-битом:
1775/rwxrwxr-t

При одновременной установке g и t битов на папку, наследование распростараняется только на g-бит
sudo chmod go+st Fold4
stat Fold4
3775/rwxrwsr-t

mkdir Fold4/Test_gt
stat Fold4/Test_gt
2775/rwxrwsr-x
